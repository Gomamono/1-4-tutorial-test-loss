# 1-4-tutorial-test-loss
応用確率統計において作成した，「クラス分類モデルの訓練方法 - Pytorch Tutorial」におけるCPU・GPU間の損失比較用コード．

当該チュートリアル
- [cifar10_tutorial.ipynb](https://colab.research.google.com/github/pytorch/tutorials/blob/gh-pages/_downloads/4e865243430a47a00d551ca0579a6f6c/cifar10_tutorial.ipynb)
- [1_4_cifar10_tutorial_jp.ipynb](https://colab.research.google.com/github/YutaroOgawa/pytorch_tutorials_jp/blob/main/notebook/1_Learning%20PyTorch/1_4_cifar10_tutorial_jp.ipynb)

## 使用方法
1. チュートリアルの最下部にセルを追加し，1-4-tutorial-test-loss.py の内容を全て貼り付ける．
2. チュートリアル中の「[3] 損失関数とオプティマイザの定義」まで実行する．
3. 貼り付けたコードを実行する．

※ モデル訓練をCPU→GPUの順で繰り返し試行するため，時間がかかることに注意．
